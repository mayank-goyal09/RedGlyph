## ğŸš€ Run Locally

```bash
# 1. Clone the repo

cd RedGlyph

# 2. Create virtual environment

.venv\Scripts\activate

# 4. Add your API key

echo "EMAIL_APP_PASSWORD=your_app_password" >> .env

# 6. Open in browser
# http://localhost:7860/app
```
